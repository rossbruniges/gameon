Category Prizes

Best Hackable Game (Non Stop Hacking)

Awarded to the games with the highest "hackability" score. Games that let their players remix mechanics, fork code or use assets from the web in order for them to create their very own version of an existing game.

Best Learning Game (Head of Class)

Awarded to the games that deepen an understanding of a concept or help a player to master a set of skills.
